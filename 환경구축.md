1. Node.js 설치
2. 클로드 데스크탑 설치: 설치를 위해서는 설정-시스템-고급-개발자용-개발자모드를 ON으로 설정해야 클로드 데스크탑 설치 가능
3. MS VS Code 설치
4. MS VS code 환경 설정(VS Code 환경설정.md)
5. MS VS code 사용자 환경 설정(VS Code 윈도우용 User Setting)
6. 폴더를 다음과 같이 생성(C:\project\my_mcp)
7. MS VS code 실행 - File > Open Foler > 위의 경로 선택 (이후 실습은 C:\project\my_mcp 에서 실행)
8. MS VS Code 에서 터미널 실행 (Ctrl + Shift + ` 단축키를 누르거나 메뉴 Terminal- New Terminal 클릭) 
9. uv 설치(아래 내용 참조)
10. uv 가상환경에서 python 설치 또는 특정 버전 선택
11. fastmcp 설치(fastMCP.md 파일 참조)
12. API Key 설치시 가급적 환경변수에 등록하고 불러오기
    


### 환경설정시 권한문제가 있는 경우
```
Set-ExecutionPolicy RemoteSigned -Scope CurrentUser -Force
```

### uv 설치 (파이썬의 여러버전을 선택하여 사용할 수 있도록 해주는 도구)
```
# On macOS and Linux.
curl -LsSf https://astral.sh/uv/install.sh | sh

# On Windows.
powershell -ExecutionPolicy Bypass -c "irm https://astral.sh/uv/install.ps1 | iex"

# With pip.
pip install uv

# With pipx.
pipx install uv

# With Homebrew(MAC OS).
brew install uv

# With Pacman.
pacman -S uv
```

### uv 정상설치 확인
```
uv
```

```
uv --version

```

```

### uv venv : uv를 활용하여 가상환경 생성, 특정 버전 설정값을 인자로 줄 수 있음
```
# 가상환경 만들기
uv venv

# 가상환경 python 특정 버전 사용
uv venv --python 3.14.2

# 가상환경 고정
uv python pin 3.13.2

# 가상환경 활성화
.venv
또는 
.venv\Scripts\activate

# 가상환경 비활성화

deactivate
```

```

### uv의 유용한 기타 명령어
- uv init: 새로운 파이썬 프로젝트를 생성함 (프로젝트 초기화)
- uv add: 프로젝트의 종속성을 추가함
- uv remove: 프로젝트의 종속성을 삭제함
- uv sync: 프로젝트의 종속성을 환경과 동기화함
- uv lock: 프로젝트의 종속성에 대한 잠금 파일을 생성함
- uv run: 프로젝트 환경에서 명령어를 실행함
- uv tree: 프로젝트의 종속성 트리를 확인할 수 있음
- uv build: 프로젝트를 배포 아카이브로 빌드함
- uv publish: 프로젝트를 패키지 인덱스에 게시함

### uv 사용예:
```
# 프로젝트 초기화
uv init

# 만약 error: Failed to discover parent workspace; 가 발생했다면 --no-workspace 옵션을 넣어주어 프로젝트를 생성해줍니다.
uv init --no-workspace

# 버전을 선택하고 싶다면, --python 옵션으로 파이썬 버전 선택
uv init --no-workspace --python 3.12

# 의존성 추가
uv add fastapi httpx

# 개발 환경 의존성
uv add --dev ruff

# 설치된 의존성 리스트
uv list

# 가상환경에서 의존성 파일들 동기화
uv sync

# 실행 가능한 패키지 실행하기
uv run pytest .

# 셀프 업데이트
uv self update
```

### 윈도우에서 API 환경 변수 설정하기

1. 터미널에서 설정하기
```
# CMD
set OPENAI_API_KEY=sk-ant-api123

# 파워셀
$env:OPENAI_API_KEY="sk-ant-api123"

# 터미널에서 영구 설정하기
setx OPENAI_API_KEY="sk-ant-api123"
```
2. UI에서 설정하기
1. 시작메뉴에서 '환경변수' 검색
2. '시스템환경 변수 편집'을 선택후 오픈
3. 시스템속성-[환경변수] 버튼 클릭
4. [새로 만들기] 버튼 클릭
5. 변수이름(N): OPENAI_API_KEY
6. 변수 값(V): sk-ant-api123 입력후 확인 버튼 클릭
7. 환경변수가 잘 설정되었는지 터미널에서 아래 내용 입력 확인
   ```
   echo %OPENAI_API_KEY%
   # 출력값
   sk-ant-api123
   ```
### Python에서 환경변수 사용하기
```

print(OPENAI_API_KEY)
```
