### VS Code 사용자 설정하기
1. Ctrl+Shift+P 를 눌러 커맨드 팔레트를 열고 'user settings' 입력
2. Preferences: Open User Settings (JSON) 선택
3. VS Code(Window용) User settings.json 파일 내용을 복사해서 기존 내용을 지운후 붙여넣기
4. File-Save (Ctrl+S) 선택해 저장

### VS Code 유용한 확장 프로그램(extensions) 설치
- 왼쪽 메뉴바에서 Extension 아이콘 클릭후 검색어 입력란에
- 'python'(Microsoft 개발) 입력후 install 버튼 클릭
- Black Formatter (소스코드의 포매팅을 자동으로 완성해 주는 프로그램) 설치
- Excel Viewer(엑셀 파일을 VS Code에서 바로 열어서 확인 가능)
- vscode-pdf (PDF파일을 VS Code에서 바로 열어서 확인 가능)
